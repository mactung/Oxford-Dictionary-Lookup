# Oxford Dictionary Lookup - Extension v0.1.0

## Store Listing Description

**Title:** Oxford Dictionary Lookup & Vocabulary Builder

**Short Description:**
Select text to view definitions from Oxford Learner's Dictionary, save words, and master them with spaced repetition quizzes on your New Tab.

**Detailed Description:**

Transform your browsing experience into a language learning journey. Oxford Dictionary Lookup allows you to instantly check definitions of any word on the web and continuously build your vocabulary.

**Key Features:**

*   **Instant Lookup**: Double-click or select any word on a webpage to get accurate definitions, phonetics, and audio from the Oxford Learner's Dictionary.
*   **Smart Vocabulary Builder**: Automatically save looked-up words to your personal library.
*   **Spaced Repetition System (SRS)**: Master vocabulary efficiently with our built-in quiz system that schedules reviews at optimal intervals.
*   **Random Practice Mode**: Stay sharp with occasional pop-up quizzes while you browse. Now with customizable intervals (15 mins, 30 mins, 1 hour+) to fit your learning pace.
*   **Beautiful New Tab Dashboard**:
    *   Track your learning progress with visual stats.
    *   Daily Practice Sessions for words due for review.
    *   **Quick Access**: View and access your favorite bookmark folders directly from the new tab.
*   **Audio Pronunciation**: Listen to correct British and American pronunciations.

### Supported Quiz Types
The extension provides diverse exercise formats to help you master words from multiple angles:

**Daily Practice (New Tab)**:
*   **Meaning**: Choose the correct definition for a given word.
*   **Spelling**: Type the correct word for a given definition (letters persist in boxes for better feedback).
*   **Audio-to-Word**: Listen to the pronunciation and identify the word.
*   **Fill in the Blank**: Choose the correct word to complete an example sentence.
*   **IPA**: Identify the correct phonetic transcription.

**Random Practice (Popup)**:
*   **Meaning**: Quick checks for definitions.
*   **IPA**: Quick checks for pronunciation.
*   **Spelling**: Type the correct word for a given definition.
